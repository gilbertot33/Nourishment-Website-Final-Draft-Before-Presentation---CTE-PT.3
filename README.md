# 🌱 Nourishment Website

**Nourishment** installs AI‑supported, solar‑powered micro‑farms across the Bronx so schools, housing sites, and community centers can grow fresh vegetables year‑round.  
This website introduces our mission, services, and interactive tools for partners and funders.

---

## 🚀 Purpose

The site’s goal is to **recruit Bronx partners and funders** for pilot installations and demonstrate how Nourishment’s technology strengthens local food security.

Visitors can:
- Learn about our micro‑farm technology
- Estimate pilot costs
- Submit partnership requests
- Explore our impact metrics

---

## 🧩 Pages Overview

| Page | Description |
|------|--------------|
| **Home** | Mission statement, differentiators, and impact highlights |
| **Services** | Details of micro‑farm units and a live pricing calculator |
| **About** | Company story, project phases, and team roles |
| **Contact** | Interactive booking form with validation |

---

## ⚙️ Interactive Features

### 1. Pricing Calculator
- Calculates startup and monthly costs based on user selections.
- Uses JavaScript event listeners and DOM manipulation to update results dynamically.

### 2. Booking Form
- Validates organization name, email, and date.
- Displays success or error messages without page reload.

---

## 🧠 Tech Stack

- **HTML5** — Semantic structure  
- **CSS3** — Responsive design using Flexbox and Grid  
- **JavaScript (ES6)** — DOM manipulation, event handling, and form validation
